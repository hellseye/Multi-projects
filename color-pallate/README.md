<p>This project generates random color palettes and allows users to copy hex values with a single click.</p>
<h2>⚙️ Features</h2>
<ul>
    <li>Generate random color palettes</li>
    <li>Display hex color codes</li>
    <li>Click any color to copy its hex value</li>
    <li>Shuffle button to generate new palettes</li>
</ul>
<h2>🛠️ Technologies Used</h2>
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
    <li>Clipboard API</li>
</ul>
<h2>📂 Files</h2>
<ul>
    <li>index.html</li>
    <li>style.css</li>
    <li>script.js</li>
</ul>
<h2>▶️ How to Run</h2>
<p>Open <strong>index.html</strong> in your browser.</p>
